# 開発者仕様書（仮）
## 概要
本システムは，アイドルマスターのキャラクター情報を
Webブラウザ上で一覧・追加・編集・削除（CRUD）できる管理アプリケーション である．
Node.js（Express）を利用したサーバサイドアプリケーションであり，学習目的のため
「データ構造の扱い」「ルーティング」「テンプレート処理」を理解することを目的としている．

以下のような階層構造を持ち、ユーザは階層を辿りながら情報にアクセスする。
・シリーズ → ユニット → メンバー
